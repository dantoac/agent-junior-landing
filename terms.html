What AgenticJunior does not do

AgenticJunior deliberately excludes shell access and browser automation. This is a security design decision, not a limitation we plan to remove. In a multi-tenant environment where the assistant acts on your behalf, these capabilities would create unacceptable risk.

The assistant can call external APIs via the http_fetch tool (limited to approved hosts configured per skill). It cannot execute arbitrary code or control a browser.
